Bu proje, C# Windows Form ortamında hazırlanmıştır.Kullanıcı bir TextBox’a sayı girer,  butona bastığında girilen sayının karesi hesaplanır.
Eğer kullanıcı geçersiz bir giriş yaparsa (örneğin harf girerse), program hata yakalama (try-catch) yapısı sayesinde kullanıcıya uygun bir uyarı mesajı gösterir.
